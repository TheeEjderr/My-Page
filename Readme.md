## 🚀 Getting Started

### View Online
Simply visit: https://theeejderr.github.io/My-Page/

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/TheeEjderr/My-Page.git
   ```

2. Navigate to the directory:
   ```bash
   cd My-Page
   ```

3. Open in your browser:
   - Double-click `index.html` or
   - Use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```
